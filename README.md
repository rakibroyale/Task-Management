# Task-Management
A lightweight, fully responsive task management front-end built with pure HTML, CSS, and JavaScript. Includes pages for login, dashboard, task listing, creation, and editing. Designed without frameworks, with modular code.


A clean, fully responsive task management front-end built entirely with vanilla HTML, CSS, and JavaScript. This project is a static, framework-free conversion, designed for simplicity and portability.

The interface includes all essential task management features—authentication UI, dashboard, task listing, task creation, editing, and error pages—implemented using modular JavaScript, semantic HTML, and scalable CSS.

The project is structured to work seamlessly with any backend. All dynamic operations (login, CRUD actions) include clear API placeholders, allowing backend developers to plug in their endpoints quickly.

Key Features
- Fully static, framework-free architecture  
- Responsive and modern UI  
- Modular HTML, CSS, and JS structure  
- LocalStorage-based mock authentication  
- Clean and maintainable folder organization  
- Zero dependencies (no React, Bootstrap, jQuery, Tailwind, etc.)

Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

Ideal For
- Developers seeking a lightweight front-end to connect with a backend  
- Learning or showcasing pure HTML/CSS/JS architecture  
- Replacing heavy frameworks with minimal static pages
